## Description
루프를 돌면서 제거해야할 값과 같지 않다면 nums[ j ] 에 현재 값을 넣고, j를 +1 시킨다.

그러니까 제거해야할 대상이 아닌것들만 추려서 앞으로 이동시킨다. (그렇다고 교환하진 않는다. 문제에서 뒤의 값은 의미 없다고 명시함)



### 복잡성
- 시간: O(n)
- 공간: O(1)
